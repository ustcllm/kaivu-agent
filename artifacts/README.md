# Artifacts

This directory is for durable research outputs that should be inspectable outside the agent runtime.

Use it for:

- Experiment outputs that are worth keeping.
- AI research generated packages.
- Reports, submissions, plots, tables, and model references.
- Reproducibility bundles that may be shared or reviewed.

Do not use this directory for short-lived scratch files. Put temporary outputs under `test_artifacts/tmp/` or `.state/tmp/`.
